## About This Course
This course covers relational database design, SQL querying, and Oracle database tools. 
All assignments were written in Oracle SQL and tested using Oracle APEX.

---

## Contents

### Assignment 1: SELECT
Basic SQL SELECT statements to retrieve data from database tables.

### Assignment 2: More on SELECT
Advanced SELECT queries including filtering with WHERE, sorting with ORDER BY, 
and using DISTINCT.

### Assignment 3: Helper Functions
Using Oracle built-in functions such as UPPER, LOWER, ROUND, TRUNC, TO_DATE, 
MONTHS_BETWEEN, and more.

### Assignment 4
SQL JOIN queries — combining data from multiple tables using INNER JOIN, 
LEFT JOIN, and ANSI/ISO SQL 99 syntax.

### Assignment 5
Additional SQL practice covering subqueries and aggregate functions 
like COUNT, SUM, AVG, MIN, MAX with GROUP BY and HAVING.

### Assignment 7
Advanced SQL topics including views, sequences, synonyms, and indexes in Oracle.

### Final Project: Movie Rental Database
Designed and built a full relational database for a movie rental store using Oracle APEX.
- 6 tables: CUSTOMERS, MOVIES, ACTORS, MEDIA, STAR_BILLINGS, RENTAL_HISTORY
- Created sequences, views, synonyms, and indexes
- Inserted sample data using sequences for all primary keys
- Built with proper constraints following professor's naming conventions
